## Purpose

This guide provides best practices, stylistic conventions, and reference material for Dungeon Masters using or contributing to the AI Dungeon Master system. It ensures consistency, clarity, and genre-accuracy in all campaign content and documentation.

## Best Practices

- **Clarity:** Use clear, concise language in all documentation and campaign materials.
- **Genre Accuracy:** Adhere to D&D conventions for mechanics, lore, and narrative style.
- **Inclusivity:** Represent diverse characters, cultures, and perspectives respectfully.
- **Consistency:** Keep terminology, formatting, and cross-references in sync across all documents.
- **Improv Tools:** Use random tables, prompts, and campaign hooks to support improvisation.

## Style Conventions

- **Headings:** Use descriptive headings and subheadings for navigation.
- **Lists & Tables:** Use bullet points and tables for clarity and quick reference.
- **Code & Stat Blocks:** Use code blocks for stat blocks, mechanics, and examples.
- **Cross-Referencing:** Link to related documents and sections for context.
